problem statement: 
under the theme of green technology, we the team GreenLoop are here to create a smart waste collection system. 
Waste collection in India has improved significantly with a roughly 92% collection rate in urban areas, yet it faces challenges, as only 54% of the ~1.7 lakh tonnes of daily waste is processed, leaving many landfills.
Municipal dustbins get overfilled in localities very quickly and cause foul smell and hygiene and sanitation issues in the area. Hence creating a system which collects waste generated timely and properly reaches the dump yards is the problem statement that we are working on today.
